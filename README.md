# SCR-Bevy-Game
Simple game made with [Bevy](https://bevyengine.org/) engine.
General idea is to hit the enemy tank by shooting on a ballistic trajectory.

![](other/ss1.png)

![](other/preview1.gif)

## How to run

1) Install [Rust](https://www.rust-lang.org/tools/install)
2) In the project root execute: `cargo run --release`
3) enjoy ;)


## Game controls

A,D - move tank left, right on the map

W,S - change aiming angle

Space - shoot
